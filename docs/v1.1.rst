Introduction
------------

This project flattens nested iterables.

Typing
------

The typing conforms (not strictly) to ``mypy``.

.. container:: versionadded

    **Added in version 1.1.**

Installation
------------

To install ``iterflat``, you can use ``pip``.
Open your terminal and run:

.. code-block:: shell

    pip install iterflat

Features
--------

``iterflat.iterflat(data: Any, *, depth: SupportsIndex = 1) -> Generator[Any, None, None]``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This generator flattens nested iterables.

``iterflat.tests.test() -> unittest.TextTestResult``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This project can be tested through its ``test`` function.

.. code-block:: python

    import iterflat.tests
    iterflat.tests.test()
